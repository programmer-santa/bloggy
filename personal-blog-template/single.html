Glosario Técnico ADSO

15 términos técnicos explicados en inglés

A - Algorithm

A set of steps used to solve a problem or perform a task in programming.

B - Backend

The part of a software system that handles logic, database, and server-side processing.

C - Class

A template used in object-oriented programming to create objects with specific attributes and methods.

D - Database

An organized collection of data that can be accessed and managed electronically.

E - Entity

A real-world object or concept that has data stored in a database.

F - Framework

A collection of tools and libraries used to simplify and speed up software development.

G - Git

A version control system used to track changes in source code.

H - HTML

The standard language used to create and structure content on the web.

I - Interface

A point where two systems or software components interact or communicate.

J - JavaScript

A programming language used to create dynamic and interactive web pages.

K - Keyword

A reserved word in a programming language with a special meaning.

L - Loop

A programming structure used to repeat a set of instructions until a condition is met.

M - Model

A representation of data or business logic, often used in MVC architecture.

N - Node.js

A runtime environment that allows JavaScript to run on the server side.

O - Object

A data structure in OOP that contains properties and functions.
